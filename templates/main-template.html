Twitterplot

Twitterplot uses data from Twitter. The current data set is from the Republican debate, following several relevant hashtags. Enter search terms separated by spaces and have fun!

TwitterPlot is easy to use. Select your plot-type from the following:

  • Bar Graph: a simple numerical count of tweets mentioning the terms.
  • Pie Chart: a pie chart with percentages of tweets
  • Time Plot: tweet frequency over time (check 'total' to sum the tweets at any given time).
  • Interval Time Plot: select a start and end time for plotting a time plot.

The Interval setting is used on Time Plot and Interval Time Plot to increase/decrease resolution of the line plot

Search Terms

You can use any number of search terms in the search dictionary, including hashtags. Capitalization does not matter.

Caveats

The demo version you're using has some limitations

  • Time Plots can plot a maximum of FIVE terms (this is because of the color list I use in matplotlib).
  • The start/end time fields take '00:00' formatted times, which are hours:minutes from the beginning of the data set. This data set is approximately three hours worth of data (but it doesn't matter if you overshoot the end)
  • The data set is from the first Republican debate following a few related hashtags. So use those search terms! You might not get much for 'python' but you will get a lot for 'trump'
  • You will notice that the data drops around 3:00AM. This is unavoidable with the way the data is displayed (this is a design decision--data is not being lost).
